Projeto Celebre com classe

O que é?

O projeto celebre com classe trata de um website onde vai reunir toda a informação de um evento assim como a lista de convidados para quem organizar o evento

O porquê de este projeto?

Este projeto foi inspirado pq ao longo da criação de um evento tivemos varios problemas com a transmissão da informação sobre o tal e quen queria de participar do mesmo assim como acompanhantes, etc...
